
## Framer Motion (Motion.dev)

**Installation:**

```
npm i motion

bun i motion
```

**If any error than:**

```
npm i motion --legacy-peer-deps

bun add motion --legacy-peer-deps
```

[Go to Official Website](https://motion.dev/docs/react)
